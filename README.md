# OpenGL Kurs (NeHe Tutorials in C)

Deutsche Übersetzung und Umsetzung der klassischen [NeHe OpenGL Tutorials](https://nehe.gamedev.net/) in C.

Ursprünglich von NeHe (gamedev.net) geschrieben, bilden diese Lektionen die Grundlage für das Verständnis von OpenGL Fixed-Function-Pipeline und modernem OpenGL.

## Struktur

- `Lektion01/` bis `Lektion51_Pong/` – Die 51 NeHe-Lektionen, jede als eigenständiges Projekt
- `Breakout_V1/` – Breakout-Clone
- `Pinball_V1/` – Pinball-Clone
- `Pong_V2/` – Pong-Clone
- `Projekt51_FensterManager/` – Fenster-Manager-Demo
- `Projekt52_FensterManager/` – Erweiterter Fenster-Manager
- `Projekt52_PengerViewer/` – Penger-Viewer
- `Projekt53_PengerPBRViewer/` – PBR-Viewer
- `Vulkan01/` – Erste Vulkan-Versuche

## Build

Jede Lektion hat ihr eigenes `build/`-Verzeichnis mit Visual-Studio-Lösung (`.sln`).
